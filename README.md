react-messages
=======================

React component for showing error, warning, info messages.

## Install
`npm install react-message`

## CSS
This is a BYOC (Bring Your Own CSS) component.

Each type of message is given a class name like "message--#{messageType}"
so:

* .message--error
* .message--warning
* .message--info
